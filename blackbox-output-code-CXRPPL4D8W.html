Perjalanan Saya

Saya mulai mengenal dunia komputer dan teknologi sejak Sekolah Dasar di tahun 2010 di SD Sudimara 02. Dari kecil saya belajar otodidak tentang komputer, mulai dari mengenal software sederhana hingga mencoba membuat program kecil menggunakan QBasic. Pada masa itu, komputer masih terasa seperti gadget ajaib yang datang dari masa depan. Guru saya yang bernama Pak Ahmad selalu memberi kami tugas sederhana seperti mencetak nama menggunakan Paint atau bermain game edukasi seperti Mario Bros. Itu adalah awal yang membangkitkan rasa ingin tahu saya. Saya ingat pertama kali melihat logo Windows 95 yang muncul di layar monitor besar di kelas, dan saya bertanya-tanya bagaimana mesin itu bisa menghasilkan gambar yang begitu indah. Setelah pulang sekolah, saya sering 'ngibul' di internet cafe dekat rumah sambil belajar dari majalah komputer seperti "Chiper" atau "Komputer Kita". Belajar otodidak di usia itu sangat menantang tetapi menyenangkan; seringkali saya terjebak jam berjam mencoba kode yang tidak mau berfungsi, tapi itu membuat saya lebih sabar dan tekun. Saya juga mulai mencoba menggambar desain sederhana menggunakan Microsoft Word dan Excel untuk membuat tabel sederhana, yang menjadi pondasi kemampuan saya dalam tools office.

Saat SMP di SMP Negeri 11 pada tahun 2016, saya mulai memperdalam kemampuan saya dengan belajar HTML, CSS, dan sedikit JavaScript secara otodidak melalui internet dan buku. Era itu adalah masa di mana smartphone mulai populer, dan saya menggunakan android sederhana untuk browsing di warung net. Saya menemukan situs seperti kompas.com tutorial yang memberikan tutorial gratis tentang web development. Setiap sore setelah sekolah, saya pergi ke warnet untuk mencoba kode-kode baru. Awalnya sangat frustrasi; coding error muncul di mana-mana, tapi itu mengajarkan saya tentang debugging yang sekarang jadi skill utama. Saya juga mulai tertarik dengan hardware komputer dan jaringan; membongkar CPU lama di rumah dan mencoba mengganti RAM sambil belajar dari forum Kaskus. Di sekolah, pelajaran komputer lebih fokus pada Microsoft Office, tapi itu tidak cukup bagi saya. Saya memanfaatkan lab untuk eksperimen dengan HTML, membuat website sederhana tentang hobi saya seperti koleksi gambar kartun. Itu periode di mana saya bergabung dengan komunitas kecil para pemula programmer di Jakarta, bertukar tips melalui Yahoo Messenger. Belajar otodidak selama ini membentuk mindset saya: tidak ada guru yang lebih baik dari trial and error. Saya juga mulai belajar bahasa pemrograman seperti Pascal untuk olimpiade tingkat kabupaten, yang membuat saya kompetitif dan motivasi.

Saat ini saya masih bersekolah di SMK Prima Unggul jurusan Teknik Informatika dan menargetkan lulus pada tahun 2024. Di SMK ini, pengalaman belajar otodidak saya sangat membantu dalam memahami konteks formal. Kurikulumnya meliputi pemrograman Java dan C++, yang baru bagi saya secara teknis, tapi logikanya mirip dengan apa yang saya pelajari sendiri. Saya juga mendapatkan pembelajaran tentang database MySQL dan jaringan komputer, yang menggabungkan interest hardware saya sebelumnya. Magang di tahun ketiga membuat saya terjun ke dunia nyata, bekerja pada projek sistem informasi untuk bisnis kecil. Itu pengalaman berharga; saya belajar tentang teamwork dan deadline, sesuatu yang tidak ada di belajar otodidak. Namun, spirit otodidak tetap kuat; di malam hari, saya belajar frame seperti React melalui YouTube dan kursus dari udemy. SMK Prima Unggul memberikan infrastruktur yang mendukung inovasi, seperti laboratorium dengan komputer terbaru dan akses internet cepat, yang membuat proses belajar lebih efisien. Saya juga aktif dalam proyek-proyek ekstrakurikuler, seperti membuat aplikasi mobile sederhana untuk sekolah atau bergabung dalam kompetisi coding nasional. Masa ini adalah puncak pengembangan saya, di mana saya belajar bahwa education formal dan otodidak saling melengkapi untuk mencapai karier di bidang IT yang saya impikan.

Secara keseluruhan, perjalanan dari SD Sudimara 02 hingga SMK Prima Unggul telah membentuk saya menjadi programmer yang mandiri dan inovatif. Era awal di SD adalah fondasi rasa ingin tahu, SMP adalah periode eksplorasi otodidak intens, dan SMK adalah pengayaan formal. Tantangan terbesar adalah ketekunan dalam menghadapi kesulitan coding, tapi itu juga yang membuat saya kuat. Ke depan, saya berencana terus belajar teknologi seperti AI dan cybersecurity, mungkin kuliah di kampus teknik atau langsung kerja di startup. Saya percaya bahwa kombinasi belajar otodidak dan formal adalah kunci sukses di dunia yang berubah cepat seperti IT. Ini bukan hanya tentang skills, tapi juga tentang passion yang saya temukan sejak kecil dan terus berkembang hingga sekarang.
